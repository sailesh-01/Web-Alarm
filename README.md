# Web-Alarm

A premium, modern web application that serves as a beautiful digital clock and a fully-functional alarm system for daily life.

## Features ✨

- **Real-Time Digital Clock**: Large, highly visible HH:MM:SS AM/PM clock display accompanied by the current date.
- **Multiple Alarms**: Set and track any number of customized alarms.
- **Persistent Data**: Alarms are saved in the browser's `localStorage` so they stick around across reloads. 
- **Modern "Stitch" / MD3 UI**: Built with responsive layouts, smooth hover animations, soft drop shadows, and pill-shaped rounded button components. 
- **Dark & Light Mode Toggle**: Seamlessly switch between dark and light themes. The app remembers your preference automatically.
- **Snooze Functionality**: Quickly tap 'Snooze' when the alarm rings to delay it by another 5 minutes!
- **Alarm Tone Ringing**: Full overlay that softly loops an audible ring until specifically stopped.

## Tech Stack 💻

Powered entirely by modern core web-tech (Vanilla Web):
- **HTML5**
- **CSS3** (Fluid variables, keyframe animations, media queries)
- **JavaScript** (ES6+, DOM manipulating, Data tracking)

## How to use 🚀

1. Open up `index.html` in any web browser of your choice.
2. Ensure browser permissions allow Media/Audio autoplay (required to ring the audible alarm without an explicit preceding click in Safari/Chrome policy).
3. Set your target Hour, Minute, and AM/PM drop-down, then hit **Set Alarm**.
4. To remove an alarm, simply tap the delete bin icon.
